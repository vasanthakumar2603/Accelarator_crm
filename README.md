# AI Powered Customer Relationship Management Portal

A simple Flask-based CRM portal project with:

- Customer login page
- Customer enquiry form
- Customer chatbot
- Admin login page
- Admin dashboard
- Email notification preview
- WhatsApp message preview
- Service demand analytics

## Services
- Digital Marketing
- Gen AI
- Data Analytics
- Advanced Artificial Intelligence
